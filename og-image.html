How to export as og-image.png 1. In Chrome DevTools, set device size to exactly 1200 × 630 (Cmd/Ctrl+Shift+M, then "Responsive", enter dimensions).
2. Cmd/Ctrl+Shift+P → type "screenshot" → "Capture full size screenshot".
3. Rename the PNG to og-image.png and place in project root.
